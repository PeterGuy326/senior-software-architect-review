# 论文范文库（真实项目改编）

> 每篇 **2500 字以上** 完整范文，**直接参考背诵**，对应 [`paper-topics/`](../paper-topics/) 中的万能提纲。

## ⚠️ 使用须知

- **非真实考生作文**，由行业常见项目改编，避免查重风险
- **建议方式**：熟读背骨架 + 替换数据为自己真实项目
- 每篇包含：**摘要 · 背景 · 理论 · 实践（4 段展开）· 总结**

## 范文评分目标

- **45+ 分（合格）**：骨架完整 + 理论正确 + 实践有细节
- **55+ 分（优秀）**：4 段实践均有**权衡讨论** + 量化数据 + 教训反思

## 复习用法

1. **精读 1 篇**（第一选择主题），熟记结构与句式
2. **泛读其他 4 篇**，积累**万能素材段落**（可跨主题复用）
3. **自己仿写 1 篇**，替换成自己真实项目数据
4. **考前 1 周默写骨架**，训练 120 分钟手速
