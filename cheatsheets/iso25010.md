# ISO/IEC 25010 软件质量模型（📚 综合高频易混淆考点）

## ⚠️ 易混淆警告

- **ATAM 质量属性**（教材第 8 章）：6 个（可用 / 性能 / 安全 / 可修改 / 可测试 / 易用）
- **ISO 25010**：8 个质量特性
- **ISO 9126**（旧版）：6 个（已被 25010 取代）

考试中若题干问"质量特性"、引用 ISO 标准 → 用 **25010 的 8 个**

## 一、ISO 25010 产品质量模型 · 8 大特性（31 子特性）

### 1. 功能适合性（Functional Suitability）

- 功能完备性
- 功能正确性
- 功能适当性

### 2. 性能效率（Performance Efficiency）

- 时间特性
- 资源利用性
- 容量

### 3. 兼容性（Compatibility）

- 共存性
- 互操作性

### 4. 易用性（Usability）

- 可辨识性
- 易学性
- 易操作性
- 用户错误保护
- 用户界面美观性
- 可访问性

### 5. 可靠性（Reliability）

- 成熟性
- 可用性
- 容错性
- 易恢复性

### 6. 安全性（Security）

- 保密性（Confidentiality）
- 完整性（Integrity）
- 抗抵赖性（Non-repudiation）
- 可核查性（Accountability）
- 真实性（Authenticity）

### 7. 维护性（Maintainability）

- 模块性
- 可重用性
- 易分析性
- 易修改性
- 易测试性

### 8. 可移植性（Portability）

- 适应性
- 易安装性
- 易替换性

## 二、ISO 25010 使用质量模型（5 特性）

从用户视角：

1. **有效性**：准确完整达成目标
2. **效率**：付出资源少
3. **满意度**：舒适度
4. **抗风险**：经济 / 健康 / 环境风险
5. **周境覆盖**：各种使用场景下的适用性

## 三、速记对照表

| ATAM 6 属性 | ≈ ISO 25010 特性 |
|---|---|
| 可用性 | 可靠性 → 可用性子特性 |
| 性能 | 性能效率 |
| 安全性 | 安全性（整个特性） |
| 可修改性 | 维护性 → 易修改性 |
| 可测试性 | 维护性 → 易测试性 |
| 易用性 | 易用性 |

> 记：**ATAM 偏架构师视角，ISO 偏产品视角**
