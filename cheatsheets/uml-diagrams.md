# UML 图速查

## UML 2.x 14 种图（分静态 / 动态）

### 静态结构图（7）

| 图 | 用途 | 关键元素 |
|---|---|---|
| **类图** Class | 系统静态结构 | 类、关联（关联/依赖/泛化/实现/聚合/组合） |
| **对象图** Object | 某时刻实例快照 | 对象、链接 |
| **组件图** Component | 物理组件 | 组件、接口 |
| **部署图** Deployment | 运行时硬件分布 | 节点、构件 |
| **包图** Package | 包的组织 | 包、依赖 |
| **组合结构图** Composite Structure | 类内部结构 | 部件、端口 |
| **制品图** Artifact | 物理制品 | 文件、库 |

### 动态行为图（7）

| 图 | 用途 | 关键元素 |
|---|---|---|
| **用例图** Use Case | 需求 | 参与者、用例、关系（包含/扩展/泛化） |
| **顺序图** Sequence | 时序交互 | 对象、消息、生命线 |
| **通信图** Communication | 协作交互（强调关系） | 对象、编号消息 |
| **状态图** State Machine | 对象状态变迁 | 状态、事件、动作 |
| **活动图** Activity | 业务流程 | 活动、泳道、分支 |
| **定时图** Timing | 时间约束 | 生命线、状态、时间轴 |
| **交互概览图** Interaction Overview | 活动图 + 交互图 | 控制流 + 交互 |

## 关系符号速记（类图）

```
泛化 (继承)    ——▷   空心三角形实线
实现           ----▷  空心三角形虚线
依赖           ---->  虚线箭头
关联           ——>   实线箭头
聚合 (弱)      ——◇   空心菱形
组合 (强)      ——◆   实心菱形
```

## 用例关系

| 关系 | 符号 | 含义 |
|---|---|---|
| **包含** include | `«include»` | 必选子流程（复用） |
| **扩展** extend | `«extend»` | 可选流程（在特定扩展点） |
| **泛化** | 三角形 | 父子用例 |

## 选图建议（案例题）

| 问题 | 用什么图 |
|---|---|
| 画系统功能 / 需求 | 用例图 |
| 画静态结构 | 类图 |
| 画时序交互 | 顺序图 |
| 画对象状态 | 状态图 |
| 画业务流程 | 活动图 |
| 画部署架构 | 部署图 |
| 画模块划分 | 包图 / 组件图 |
