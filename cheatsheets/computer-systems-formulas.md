# 计算机系统计算公式速查

## 一、Cache & 存储

| 公式 | 含义 |
|---|---|
| `T = h·Tc + (1-h)·Tm` | 平均访问时间（h 命中率） |
| `映射方式` | 直接映射、全相联、组相联 |
| `替换算法` | LRU / FIFO / OPT / Random |

## 二、流水线

| 公式 | 含义 |
|---|---|
| `执行时间 T = (k + n - 1)·Δt` | k 段流水线执行 n 条指令 |
| `吞吐率 TP = n / T` | 单位时间完成指令数 |
| `加速比 S = 顺序时间 / 流水线时间` | 与非流水对比 |
| `效率 E = 有效忙时 / 总面积` | 流水线利用率 |

## 三、可靠性

| 系统结构 | 可靠性 |
|---|---|
| 串联 | R = R₁·R₂·…·Rn |
| 并联 | R = 1 − (1-R₁)·(1-R₂)·…·(1-Rn) |
| N 取 K 表决 | 组合计算 |

> 串联降可靠，并联升可靠

## 四、磁盘

| 公式 | 含义 |
|---|---|
| 存取时间 = 寻道 + 旋转等待 + 传输 | 三项组成 |
| 平均旋转等待 = 旋转周期 / 2 | 随机分布 |
| 磁盘调度：FCFS / SSTF / SCAN / C-SCAN | 记选择标准 |

## 五、性能度量

| 指标 | 公式 |
|---|---|
| MIPS | 百万条指令/秒 |
| MFLOPS | 百万次浮点运算/秒 |
| 响应时间 | 请求到收到首字节时间 |
| 吞吐量 | 单位时间完成事务数 |

## 六、校验码

| 码 | 检错 | 纠错 | 备注 |
|---|---|---|---|
| 奇偶校验 | 1 位错 | 不能 | 最简单 |
| CRC | 多位错 | 不能 | 通信常用 |
| 海明码 | 多位错 | 1 位错 | 2^k ≥ m + k + 1 |

## 七、I/O 控制方式对比

| 方式 | CPU 占用 | 典型场景 |
|---|---|---|
| 程序查询 | 高 | 简单设备 |
| 中断 | 中 | 字符设备 |
| DMA | 低 | 块设备（磁盘） |
| 通道 | 极低 | 大型机 |
