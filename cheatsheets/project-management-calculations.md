# 项目管理计算题速查（📚 综合知识 2–3 题）

## 一、关键路径法（CPM）

**定义**：从开始到结束**耗时最长**的路径（决定项目工期）

### 节点时间参数

- **最早开始** ES = max(前置活动 EF)
- **最早完成** EF = ES + 工期
- **最晚完成** LF = min(后续活动 LS)
- **最晚开始** LS = LF - 工期
- **总时差** TF = LS - ES（非关键路径可拖延时间）
- **自由时差** FF = 后继 ES - 本活动 EF

### 关键路径特征

- 总时差 = 0 的活动组成的路径
- **可能有多条**

## 二、PERT 三点估算

**期望工期** E = (乐观 + 4×最可能 + 悲观) / 6

**方差** σ² = ((悲观 - 乐观) / 6)²

### 典型题

> A = 4, M = 6, B = 8，求期望工期
>
> E = (4 + 24 + 8) / 6 = 6

## 三、挣值管理（EVM）

### 三个基础值

| 缩写 | 全称 | 含义 |
|---|---|---|
| **PV** | Planned Value | 计划值（原定到某日应完成的工作预算） |
| **EV** | Earned Value | 挣值（实际完成工作的预算） |
| **AC** | Actual Cost | 实际成本 |

### 偏差与指数

| 公式 | 含义 |
|---|---|
| **SV** = EV - PV | 进度偏差（正数=超前） |
| **CV** = EV - AC | 成本偏差（正数=节约） |
| **SPI** = EV / PV | 进度绩效指数（> 1 超前） |
| **CPI** = EV / AC | 成本绩效指数（> 1 节约） |

### 预测

| 公式 | 含义 |
|---|---|
| **BAC** | 完工预算 |
| **EAC** = BAC / CPI | 完工估算（典型性差异） |
| **ETC** = EAC - AC | 完工尚需成本 |

### 典型题

> 计划 10 万，到期应完成 50%（PV = 5 万），实际完成 40%（EV = 4 万），实际花了 5.5 万（AC = 5.5 万）
>
> - SV = 4 - 5 = -1（**落后**）
> - CV = 4 - 5.5 = -1.5（**超支**）
> - SPI = 0.8 / CPI = 0.73
