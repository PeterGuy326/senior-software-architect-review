# 综合知识英语阅读题套路（📚 每年固定 5 题 = 5 分）

## 一、题型特点

- **每年固定 5 题**
- **位置**：综合知识试卷**最后 5 题**
- **形式**：一篇 IT 英语短文 + 5 个选词填空（每空 4 选 1）
- **主题**：云计算、AI、大数据、安全、敏捷、DevOps、架构设计等

## 二、答题策略

### 策略 1：不要死磕全文

- **先看选项**：4 个选项的**词性**和**词义**
- **精读空格上下句**：80% 情况下**单句就能解**

### 策略 2：词性优先

- 空后是名词 → 空格位置是形容词 / 冠词 / 动词
- 空后是动词 → 空格位置是主语 / 副词
- **先排错词性**再看词义

### 策略 3：词义推断

- **主题一致**：选和短文主题相关的词
- **逻辑一致**：并列 / 转折 / 因果要能连通

## 三、高频词汇（按主题）

### 架构 & 系统

- architecture, component, module, interface, layer, tier, framework
- scalability, availability, reliability, maintainability, performance
- coupling, cohesion, abstraction, encapsulation

### 云计算 / 云原生

- IaaS / PaaS / SaaS / FaaS, container, orchestration, Kubernetes, serverless
- virtualization, hypervisor, elasticity, multi-tenancy

### 微服务 & 分布式

- microservice, monolithic, API gateway, service mesh
- consistency, partition, replication, sharding, load balancing

### 数据 & AI

- dataset, training, model, inference, neural network
- big data, streaming, batch, ETL, data lake, warehouse

### 安全

- authentication, authorization, encryption, certificate
- vulnerability, threat, attack, intrusion, firewall

### 敏捷 & DevOps

- agile, Scrum, iteration, sprint, backlog, retrospective
- CI/CD, pipeline, deployment, automation, monitoring

## 四、高频连词 / 逻辑词

| 逻辑 | 英文 |
|---|---|
| 并列 | and, as well as, in addition |
| 转折 | but, however, nevertheless, yet |
| 因果 | because, since, due to, as a result, therefore |
| 举例 | for example, such as, including |
| 对比 | compared to, unlike, while, whereas |
| 时间 | before, after, when, while, once |

## 五、真题常考句型

1. **条件句**：`If ... , then ...`
2. **被动语态**：`is designed to / is used for / is based on`
3. **定语从句**：`which / that / who`
4. **非谓语**：`-ing / -ed / to do` 作定语

## 六、应试节奏

- **综合知识总时长**：120–150 min
- **75 题平均每题 2 min**
- **英语 5 题总时长 10–15 min**
- **若时间紧**：英语最后做，前 70 题先拿分

## 七、万一卡壳救急

- 靠前后文词性排除 2 个错项
- 剩下 2 选 1，选**更贴主题**的
- **4 个选项不选同一个字母**（避免统计偏差）

## 八、备考建议

- **读 5 年真题英语原文**，积累 IT 词汇
- **每周 1 篇英文技术博客**（Martin Fowler / ThoughtWorks）
- **不用刷四六级**，软考英语比四级简单
