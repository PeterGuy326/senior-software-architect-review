# 知识产权与标准化（📚 选择题 2–3 题，**必考**）

## 一、知识产权分类

| 类型 | 保护期 | 对象 | 法律依据 |
|---|---|---|---|
| **著作权** | 作者终生 + 死后 50 年（法人作品 50 年） | 软件、文字、音乐、美术 | 著作权法 |
| **专利权** | 发明 **20 年** / 实用新型 **10 年** / 外观设计 **15 年** | 发明创造 | 专利法 |
| **商标权** | **10 年**（可续展） | 商标、服务标 | 商标法 |
| **商业秘密** | 不公开即永久 | 技术 + 经营信息 | 反不正当竞争法 |

## 二、计算机软件保护条例（重点）

- **软件著作权**：自软件开发完成之日起自动产生
- **保护期**：自然人 **作者终生 + 死后 50 年**；法人 **50 年**（自首次发表起算）
- **不保护思想，只保护表达**
- **合作开发**：共有著作权（未约定按各自完成部分单独享有 + 整体共有）
- **职务作品**：一般著作权归**单位**（涉及本职工作或利用单位物质技术条件）
- **委托开发**：按合同约定，**未约定归受托方**

## 三、典型情景题

| 情景 | 归属 |
|---|---|
| 员工本职工作开发软件 | 单位 |
| 员工业余时间自主开发 | 个人 |
| 委托开发无约定 | 受托方 |
| 委托开发约定归委托方 | 委托方 |
| 合作开发无约定 | 共有 |

## 四、侵权判定

- **原则**：思想不保护，表达才保护
- **实质性相似**：判定侵权关键
- **接触 + 相似**：完整举证链

## 五、标准化

### 标准分类

| 层级 | 代号 | 示例 |
|---|---|---|
| 国际标准 | ISO / IEC / ITU | ISO 9001, ISO 25010 |
| 国家标准 | GB（强制）/ GB/T（推荐） | GB/T 38674 人工智能算法 |
| 行业标准 | HB / YY / SJ | — |
| 地方标准 | DB | — |
| 团体/企业标准 | T/ / Q/ | — |

### 重要标准（必记）

| 标准 | 内容 |
|---|---|
| **ISO 9000** 系列 | 质量管理 |
| **ISO/IEC 25010** | 软件产品质量模型（8 质量特性） |
| **ISO/IEC/IEEE 42010** | 架构描述 |
| **CMMI** | 能力成熟度（5 级） |
| **GB/T 22080** = ISO 27001 | 信息安全管理 |
| **GB/T 22239** | 等级保护基本要求（等保 2.0） |

## 六、ISO/IEC 25010 软件质量模型 8 个特性（重要）

1. 功能适合性（Functional Suitability）
2. 性能效率（Performance Efficiency）
3. 兼容性（Compatibility）
4. 易用性（Usability）
5. 可靠性（Reliability）
6. 安全性（Security）
7. **维护性**（Maintainability）
8. **可移植性**（Portability）

> 注：软考 6 质量属性是 **ATAM 的六元组**（可用/性能/安全/可修改/可测试/易用），与 ISO 25010 的 8 质量特性**不同**，考试常混淆，要区分！
