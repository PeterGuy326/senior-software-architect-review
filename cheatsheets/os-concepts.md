# 操作系统核心考点速查（📚 选择题高频）

## 一、进程与线程

| 维度 | 进程 | 线程 |
|---|---|---|
| 资源 | 独立地址空间 | 共享进程资源 |
| 切换开销 | 大 | 小 |
| 通信 | IPC（管道/共享内存/消息/信号/Socket） | 直接共享变量 |

## 二、进程调度算法

| 算法 | 全称 | 特点 |
|---|---|---|
| **FCFS** | First Come First Served | 先来先服务 |
| **SJF** | Shortest Job First | 短作业优先（可能饿死长任务） |
| **RR** | Round Robin | 时间片轮转 |
| **Priority** | 优先级调度 | 可抢占 / 不可抢占 |
| **多级反馈队列** | | 综合最优 |

## 三、死锁

### 四个必要条件（全记）

1. **互斥**（Mutual Exclusion）
2. **占有且等待**（Hold and Wait）
3. **不可剥夺**（No Preemption）
4. **循环等待**（Circular Wait）

### 死锁处理

| 方法 | 思路 |
|---|---|
| 预防 | 破坏 4 条件之一 |
| 避免 | **银行家算法**（安全序列） |
| 检测 | 资源分配图 |
| 解除 | 剥夺 / 终止进程 |

### 死锁资源数计算公式

> N 个进程，每个最多需 M 个资源，要保证不死锁，资源总数 ≥ `N × (M-1) + 1`

## 四、内存管理

### 分页 vs 分段 vs 段页式

| 方式 | 地址结构 | 特点 |
|---|---|---|
| 分页 | 页号 + 页内偏移 | 固定大小，消除外碎片 |
| 分段 | 段号 + 段内偏移 | 按逻辑划分，有外碎片 |
| 段页式 | 段号 + 页号 + 页内偏移 | 结合两者 |

### 页面置换算法（**必考**）

| 算法 | 全称 | 缺点 |
|---|---|---|
| **OPT** | 最优置换 | 理论最优，无法实现 |
| **FIFO** | 先进先出 | **Belady 异常**（页框增加反而缺页增多） |
| **LRU** | 最近最少使用 | 开销大 |
| **LFU** | 最不经常使用 | 历史影响大 |
| **CLOCK** | 时钟算法 | LRU 近似 |

## 五、文件系统

- FAT / NTFS / ext4 / XFS
- 索引结构：直接 / 一级间接 / 二级 / 三级（UNIX inode）
- 磁盘调度：FCFS / SSTF / SCAN / C-SCAN / LOOK

## 六、典型计算题

### 题型 1：缺页率

给定页面引用序列 + 页框数 + 算法，求缺页次数 / 命中率

### 题型 2：磁盘存取时间

存取时间 = 寻道时间 + 旋转等待 + 传输时间

平均旋转等待 = 旋转周期 / 2

### 题型 3：银行家算法

给 Max / Allocation / Available，判能否分配 / 找安全序列
