# GoF 23 设计模式速查

## 分类口诀

> **创建 5、结构 7、行为 11**

## 一、创建型（5）— 关注对象创建

| 模式 | 意图 | 典型场景 |
|---|---|---|
| **单例** Singleton | 保证唯一实例 | 配置管理、线程池 |
| **工厂方法** Factory Method | 子类决定实例化类 | 插件、框架扩展点 |
| **抽象工厂** Abstract Factory | 创建相关对象族 | 跨平台 UI、数据库方言 |
| **建造者** Builder | 分步构造复杂对象 | SQL Builder、组装报表 |
| **原型** Prototype | 克隆对象 | 性能敏感对象创建 |

## 二、结构型（7）— 关注对象组合

| 模式 | 意图 | 典型场景 |
|---|---|---|
| **适配器** Adapter | 接口转换 | 旧系统对接 |
| **桥接** Bridge | 抽象与实现解耦 | 驱动 + 设备 |
| **组合** Composite | 树形层次 | 文件系统、UI 树 |
| **装饰** Decorator | 动态添加职责 | Java IO 流 |
| **外观** Facade | 统一入口 | 简化复杂子系统 |
| **享元** Flyweight | 共享细粒度对象 | 连接池、字符池 |
| **代理** Proxy | 代表控制访问 | RPC、AOP、延迟加载 |

## 三、行为型（11）— 关注对象交互

| 模式 | 意图 | 典型场景 |
|---|---|---|
| **责任链** Chain of Responsibility | 链式处理请求 | Filter、审批流 |
| **命令** Command | 请求封装为对象 | 撤销重做、任务队列 |
| **解释器** Interpreter | 语法解释 | SQL 解析、规则引擎 |
| **迭代器** Iterator | 顺序访问集合 | 集合遍历 |
| **中介者** Mediator | 集中协调 | MVC 控制器、聊天室 |
| **备忘录** Memento | 保存恢复状态 | 事务回滚 |
| **观察者** Observer | 一对多通知 | 事件、MQ 订阅 |
| **状态** State | 状态切换行为 | 订单状态机 |
| **策略** Strategy | 算法可替换 | 支付方式、压缩算法 |
| **模板方法** Template Method | 骨架延迟步骤 | 钩子方法 |
| **访问者** Visitor | 结构—操作解耦 | AST 遍历 |

## 记忆口诀

- **创建 5**：工抽建原单（工厂方法、抽象工厂、建造者、原型、单例）
- **结构 7**：适桥组装外享代（适配器、桥接、组合、装饰、外观、享元、代理）
- **行为 11**：责命解迭中备观状策模访

## 易混淆对比

| 对比 | 核心差异 |
|---|---|
| 工厂方法 vs 抽象工厂 | 前者单产品，后者产品族 |
| 代理 vs 装饰 | 代理控制访问，装饰增加职责 |
| 策略 vs 状态 | 策略由客户选，状态内部自切 |
| 适配器 vs 外观 | 适配器转换现有接口，外观简化新入口 |
