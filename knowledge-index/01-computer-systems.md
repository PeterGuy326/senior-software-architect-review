# 知识点 01 · 计算机系统基础

## 一、核心概念

- **Cache 命中率**：h = N_cache / (N_cache + N_main)；**平均访问时间** t = h·t_cache + (1-h)·t_main
- **流水线**：周期 = max(各段耗时)；**吞吐率** = n / [(k + n - 1) × Δt]；**加速比** S = T_顺序 / T_流水
- **CRC**：发送端用**生成多项式 G(x)** 做除法，余数附在末尾；接收端能整除则无错
- **海明码**：校验位数 k 满足 **2^k ≥ n + k + 1**
- **可靠性串并联**：串联 R = ΠRi；并联 R = 1 - Π(1-Ri)
- **大小端**：大端高字节在低地址；小端反之

## 三、典型例题

### 📚 选择题 1（Cache 平均访问时间）

某 CPU 中，Cache 访问时间 2ns，主存 200ns，命中率 98%。则平均访存时间为：
- A. 2ns  B. 5.96ns  C. 200ns  D. 202ns

**答案**：B
**解析**：t = 0.98 × 2 + 0.02 × 200 = 1.96 + 4 = **5.96ns**

### 📚 选择题 2（流水线吞吐率）

4 段流水线周期分别为 2/3/4/1 ns，输入 100 条指令。吞吐率 ≈？
- A. 0.25 条/ns  B. 1/4 条/ns  C. 100/(4×4+99×4)  D. 100/(10+99×4)

**答案**：D
**解析**：流水线周期 = max = 4ns；总时间 = (4+100-1) × 4 = 412ns；吞吐率 = 100 / 412 ≈ 0.243

### 📚 选择题 3（海明码校验位数）

数据位 7，需要几位校验码？
- A. 3  B. 4  C. 5  D. 6

**答案**：B
**解析**：2^4 = 16 ≥ 7 + 4 + 1 = 12 ✓；2^3 = 8 < 11 ✗

### 🎯 案例填空：系统可靠性

三模块串联，R1=0.9, R2=0.95, R3=0.85，求系统可靠度。

**答案**：R = 0.9 × 0.95 × 0.85 = **0.72675**

若每模块采用**双机并联热备**，新可靠度？
**答案**：每模块 R' = 1 - (1-Ri)² → R1'=0.99, R2'=0.9975, R3'=0.9775 → R = **0.965**
