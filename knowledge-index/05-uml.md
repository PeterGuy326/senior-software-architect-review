# 知识点 05 · UML 建模

## 一、核心概念

- **结构图 7 种**：类 / 对象 / 组件 / 部署 / 包 / 组合结构 / Profile
- **行为图 7 种**：**用例 / 活动 / 状态 / 时序 / 通信 / 定时 / 交互概览**
- **类间 6 关系**：
  - **泛化**（继承）▷—— 空心三角实线
  - **实现**（接口）▷╌╌ 空心三角虚线
  - **关联** ——
  - **聚合** ◇——（整体-部分，**可分离**）
  - **组合** ◆——（整体-部分，**共存亡**）
  - **依赖** ╌> 虚线箭头
- **用例关系**：`<<include>>`（必须）/ `<<extend>>`（可选）/ 泛化
- **时序图**：生命线、激活条、同步/异步消息、loop/alt/opt 片段

## 三、典型例题

### 📚 选择题 1（类关系）

"订单包含订单明细，订单删除时明细一并删除"属于哪种关系？
- A. 关联  B. 聚合  C. **组合**  D. 依赖

**答案**：C（生命周期绑定 = 组合）

### 📚 选择题 2（用例关系）

下单用例无论如何都要调用"校验库存"，应用：
- A. **<<include>>**  B. <<extend>>  C. 泛化  D. 关联

**答案**：A（必须调用 = include）

### 📚 选择题 3（时序图）

时序图中实心箭头代表：
- A. 异步消息  B. **同步消息**  C. 返回  D. 创建

**答案**：B

### 🎯 案例填空：类图关系判定

- 公司"拥有"员工，员工可辞职跳槽 → **聚合**
- 人与身份证（人亡身份证作废）→ **组合**
- Service 注入 DAO → **依赖 / 关联**
- `class Dog extends Animal` → **泛化**
- `class List implements Iterable` → **实现**
