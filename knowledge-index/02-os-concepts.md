# 知识点 02 · 操作系统

## 一、核心概念

- **进程三态**：就绪 ⇄ 运行 ⇄ 阻塞（运行→阻塞是主动，阻塞→就绪被动）
- **调度算法**：FCFS / SJF（短作业优先）/ RR（时间片）/ 优先级 / 多级反馈
- **死锁 4 必要条件**：**互斥 / 占有并等待 / 不剥夺 / 循环等待**
- **死锁处理**：预防（破坏 4 条件之一）/ 避免（**银行家算法**）/ 检测 + 恢复 / 鸵鸟
- **PV 操作**：P = 申请资源（sem--，<0 阻塞）；V = 释放（sem++，≤0 唤醒）
- **分页 vs 分段**：页定长（物理）；段变长（逻辑）；**段页式** = 段内再分页
- **页面置换**：FIFO / LRU / **Clock** / OPT（最优，理论上限）

## 三、典型例题

### 📚 选择题 1（死锁必要条件）

下列**不是**死锁必要条件的是：
- A. 互斥  B. 占有并等待  C. **可抢占**  D. 循环等待

**答案**：C
**解析**：4 必要条件是 **不剥夺**（即不可抢占），C 描述相反

### 📚 选择题 2（进程状态转换）

进程从"运行"直接转到"就绪"可能由于：
- A. I/O 请求  B. **时间片耗尽**  C. I/O 完成  D. 等待信号量

**答案**：B
**解析**：时间片到被剥夺 CPU → 就绪；A/D 是运行→阻塞；C 是阻塞→就绪

### 📚 选择题 3（页面置换）

进程访问页面序列 1,2,3,4,1,2,5,1,2,3，物理块 3 块，**LRU** 缺页次数？
- A. 7  B. **8**  C. 9  D. 10

**答案**：B
**解析**：逐步：1(缺)→2(缺)→3(缺)→4(缺,淘汰1)→1(缺,淘汰2)→2(缺,淘汰3)→5(缺,淘汰4)→1(命中)→2(命中)→3(缺,淘汰5) = **8 次**

### 🎯 案例填空：PV 同步

生产者-消费者，缓冲区 n=10：
```
生产者：P(empty); P(mutex); 生产; V(mutex); V(full);
消费者：P(full);  P(mutex); 消费; V(mutex); V(empty);
```
信号量初值：**empty = 10, full = 0, mutex = 1**

常见陷阱：**mutex 和资源信号量不能交换顺序**，否则死锁。
