# 知识点 09 · 软件度量

## 一、核心概念

### 功能点 FP

- **外部输入 EI** / **外部输出 EO** / **外部查询 EQ** / **内部逻辑文件 ILF** / **外部接口文件 EIF**
- **UFP**（未调整功能点）= Σ 各类×权重
- **FP** = UFP × (0.65 + 0.01 × Σ调整因子)

### McCabe 环路复杂度

- V(G) = **E - N + 2**（E 边数，N 节点数）
- 或 V(G) = **P + 1**（P 判定节点数，即 if / while / case）
- 建议阈值：**≤ 10**（超过 10 则复杂度过高）

### 耦合（由低到高）

1. **非直接**（无耦合）
2. **数据耦合**（传递基本数据）
3. **标记耦合**（传递数据结构）
4. **控制耦合**（传递控制标志）
5. **外部耦合**（共享外部环境）
6. **公共耦合**（共享全局）
7. **内容耦合**（直接访问内部）

### 内聚（由低到高）

1. **偶然** → 2. **逻辑** → 3. **时间** → 4. **过程** → 5. **通信** → 6. **顺序** → 7. **功能**（最强）

**设计目标**：**高内聚、低耦合**

## 三、典型例题

### 📚 选择题 1（环路复杂度）

某函数有 1 个 if、2 个 while、1 个 switch（3 分支），环路复杂度？
- A. 5  B. **7**  C. 4  D. 6

**答案**：B
**解析**：判定节点 P = 1 + 2 + (3-1) = 5；V(G) = 5 + 1 = **6**，若算 switch 整体 P = 1+2+3 = 6，V=7

### 📚 选择题 2（耦合级别）

模块 A 传递一个结构体给 B，B 只用其中 2 个字段：
- A. 数据耦合  B. **标记耦合**  C. 控制耦合  D. 公共耦合

**答案**：B

### 📚 选择题 3（内聚级别）

一个模块内部所有函数都服务于"处理登录"这一目标：
- A. 逻辑内聚  B. 过程内聚  C. **功能内聚**  D. 时间内聚

**答案**：C

### 🎯 案例填空：耦合治理

识别公共耦合（全局变量）→ 改造为**参数传递（数据耦合）**，降低耦合强度。
