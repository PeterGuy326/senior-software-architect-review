## 三种例题形式说明

- **📚 选择题**：题干 + 4 选项 + 答案 + 解析（对应科目 1）
- **🎯 案例题**：填空 / 简答，含参考答案要点（对应科目 2）
- **✍️ 论文场景**：该知识点在论文中的万能句式（对应科目 3）

## 建议复习路径

```
阶段 1：按知识点扫描  — 看"核心概念"3 分钟/个
阶段 2：做例题         — 选择题直接秒，案例题照套路
阶段 3：对应资源深读    — 哪里不熟读哪里
阶段 4：论文场景背诵    — 挑 3–5 条万能句式
```
