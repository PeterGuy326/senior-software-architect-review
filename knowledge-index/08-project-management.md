# 知识点 08 · 项目管理

## 一、核心概念

### CPM（关键路径法）

- 活动**最早**开始时间 ES = max(前置活动 EF)
- 活动**最迟**开始时间 LS = min(后续活动 LS) - 本活动时长
- **总时差** TF = LS - ES = LF - EF
- **关键路径**：TF = 0 的活动串（**工期最长**）

### PERT（三点估算）

- **期望工期** T = (T乐观 + 4·T最可能 + T悲观) / 6
- **标准差** σ = (T悲 - T乐) / 6
- **方差** σ² = [(T悲 - T乐) / 6]²

### EVM（挣值管理）

| 指标 | 公式 | 含义 |
|---|---|---|
| **PV** 计划价值 | 计划完成工作预算 | — |
| **EV** 挣值 | 实际完成工作预算 | — |
| **AC** 实际成本 | 实际花费 | — |
| **SV** = EV - PV | >0 进度超前 | |
| **CV** = EV - AC | >0 成本节约 | |
| **SPI** = EV / PV | >1 进度快 | |
| **CPI** = EV / AC | >1 成本优 | |
| **EAC** = BAC / CPI | 完工估算 | |

### 风险应对

- **规避**（避免）/ **转移**（保险）/ **减轻**（降低）/ **接受**

## 三、典型例题

### 📚 选择题 1（关键路径）

网络图 A(3)→C(5)；A(3)→B(4)→C(5)，关键路径长度？
- A. 8  B. 9  C. **12**  D. 5

**答案**：C（3+4+5 = 12 > 3+5 = 8）

### 📚 选择题 2（PERT 期望）

T乐观 = 4，T最可能 = 8，T悲观 = 18，期望工期？
- A. 8  B. **9**  C. 10  D. 11

**答案**：B
**解析**：(4 + 4×8 + 18) / 6 = 54/6 = **9**

### 📚 选择题 3（EVM 分析）

PV=100，EV=80，AC=90，判断状态：
- A. 进度超前，成本超支  B. **进度落后，成本超支**  C. 进度超前，成本节约  D. 进度落后，成本节约

**答案**：B
**解析**：SV = 80-100 = -20（落后）；CV = 80-90 = -10（超支）

### 🎯 案例填空：SPI / CPI 计算

上题中 SPI = EV/PV = **0.8**；CPI = EV/AC ≈ **0.89**；
若 BAC = 500，EAC = BAC/CPI = **562.5**，成本将超支 **62.5**。
