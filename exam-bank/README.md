## 答题套路（每题用 3 秒判断）

```
1. 读题干，圈出关键词
2. 看 4 个选项，排除明显错误 2 个
3. 剩 2 个选项 → 回到题干找最贴合定义的关键词
4. 拿不准：选"最全面"的那个，或"教材原话"那个
```
