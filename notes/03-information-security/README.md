# 03 · 信息安全技术（教材第 4 章）

## 核心考点

- 密码学基础：对称（DES、AES、IDEA）、非对称（RSA、ECC）、Hash（MD5、SHA）
- 数字签名与 PKI 体系（CA、证书、CRL）
- 安全协议：SSL/TLS、IPSec、SSH、Kerberos
- 防火墙类型（包过滤 / 状态检测 / 应用代理）、DMZ
- IDS / IPS、蜜罐
- 访问控制：DAC / MAC / RBAC / ABAC
- 等保 2.0 五级
- 国产密码算法：SM1/2/3/4/9
