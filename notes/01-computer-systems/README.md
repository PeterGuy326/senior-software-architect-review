# 01 · 计算机系统基础（教材第 2 章）

## 核心考点

- CPU 结构与指令系统（CISC / RISC）
- 存储系统层次：Cache、主存、虚存（局部性原理、替换算法 LRU/FIFO/OPT）
- I/O 控制方式（程序查询 / 中断 / DMA / 通道）
- 总线结构、可靠性公式（串联/并联/表决）
- 校验码：奇偶、CRC、海明码

## 高频计算

- Cache 命中率与平均访问时间：`T = h·Tc + (1-h)·Tm`
- 流水线吞吐率、加速比、效率
- 可靠性：串联 R = ΠRi；并联 R = 1 - Π(1-Ri)
