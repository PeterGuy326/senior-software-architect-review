# 10 · 未来信息综合技术（教材第 11 章）— 🔥 2024/2025 新增重点

## 一、云计算

- 服务模型：IaaS / PaaS / SaaS
- 部署模型：公有云 / 私有云 / 混合云 / 社区云
- 虚拟化（Hypervisor Type 1/2）vs 容器

## 二、云原生（高频新考点）

- 十二要素应用（12-Factor App）
- 容器（Docker）与编排（Kubernetes）
- Service Mesh（Istio）
- Serverless（FaaS / BaaS）
- DevOps / CI / CD
- 可观测性三支柱：Metrics、Logging、Tracing

## 三、大数据

- Lambda 架构（Batch + Speed + Serving）
- Kappa 架构（仅流处理）
- 数据湖 vs 数据仓库
- 典型组件：Hadoop / Spark / Flink / Kafka

## 四、人工智能 / AIGC（2025 新增）

- 机器学习类型：监督 / 无监督 / 强化
- 深度学习：CNN / RNN / Transformer
- AIGC 工程化：模型服务化、RAG、Prompt 工程
- AI 伦理与合规

## 五、区块链（2025 新增）

- 共识机制：PoW / PoS / PBFT / Raft
- 智能合约
- 联盟链 vs 公链 vs 私链

## 六、物联网 (IoT)

- 架构：感知层 / 网络层 / 应用层
- 边缘计算
