# 23 · 前沿技术与合规（🔥 2024/2025 新考点）

## 一、中台架构

### 三大中台

- **业务中台**：共享业务能力（用户 / 订单 / 商品 / 交易）
- **数据中台**：数据资产化、指标体系、OneID
- **技术中台**：通用技术能力（网关、MQ、监控）

### 中台 vs 微服务

| 维度 | 微服务 | 中台 |
|---|---|---|
| 目标 | 技术解耦 | 能力复用 |
| 粒度 | 细 | 更粗（业务域） |
| 治理 | 去中心 | 中心 + 下沉 |

## 二、Service Mesh

- **控制面 / 数据面分离**
- **Istio**（控制面）+ **Envoy**（数据面）
- 解决：微服务治理与业务代码耦合

### Sidecar 模式

每个服务配一个代理，接管流量治理

## 三、MLOps / AI 工程化

### 阶段

1. 数据准备（标注、清洗）
2. 模型训练
3. 模型服务化（TensorFlow Serving / TorchServe）
4. 监控（数据漂移、模型漂移）
5. 持续训练（A/B、灰度）

### AIGC 工程化（2025 新增）

- **LLM 推理部署**：vLLM / TGI
- **RAG**（检索增强生成）+ **向量数据库**（Milvus / Pinecone / Qdrant）
- **Prompt 工程** + **Agent 架构**
- **微调**（LoRA / QLoRA / P-Tuning）

## 四、区块链（2025 新增）

### 共识机制

| 机制 | 代表 | 特点 |
|---|---|---|
| **PoW** | BTC | 算力竞争，耗能 |
| **PoS** | ETH 2.0 | 权益质押 |
| **PBFT** | 联盟链 | 高吞吐、信任节点 |
| **Raft** | 私有链 | 强一致 |

### 联盟链 vs 公链 vs 私有链

- 联盟链：Hyperledger Fabric、FISCO BCOS
- 场景：供应链金融、数字证书、溯源

## 五、隐私计算

- **联邦学习**（Federated Learning）：数据不出域，模型共享
- **多方安全计算**（MPC）
- **可信执行环境**（TEE，如 Intel SGX）
- **差分隐私**

## 七、边缘计算与物联网

## 八、DevOps & CI/CD

### 核心实践

- 版本控制（Git）
- 持续集成（Jenkins / GitLab CI / GitHub Actions）
- 持续部署（ArgoCD / Flux）
- 基础设施即代码（Terraform / Ansible）
- 可观测性（Prometheus / Grafana / SkyWalking）

### DORA 四大指标

1. 部署频率
2. 变更前置时间
3. 变更失败率
4. 服务恢复时间
