# 21 · 知识产权与标准化（📚 综合 2–3 题必考）

## 一、知识产权

- 著作权：作者终生 + 死后 50 年
- 专利权：发明 20 年 / 实用新型 10 年 / 外观设计 15 年
- 商标权：10 年（可续展）
- 商业秘密：不公开即永久

## 二、计算机软件保护条例

- 软件著作权：开发完成自动产生
- 职务作品 → 单位
- 委托开发无约定 → 受托方

## 三、标准化层级

| 层级 | 代号 |
|---|---|
| 国际 | ISO / IEC / ITU |
| 国家（强制） | GB |
| 国家（推荐） | GB/T |
| 行业 | HB / YY / SJ |
| 地方 | DB |
| 团体 / 企业 | T / Q/ |

## 四、重要标准

- ISO 9000 质量管理
- **ISO/IEC 25010** 软件质量模型（8 特性）
- **ISO/IEC/IEEE 42010** 架构描述
- ISO/IEC 27001 信息安全管理
- **GB/T 22239** 等保 2.0
- CMMI 能力成熟度（5 级）
