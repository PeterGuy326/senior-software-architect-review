# 14 · 案例：面向服务架构 SOA（教材第 15 章）

## 核心概念

- 服务 = 自治、松耦合、可复用的业务功能
- **ESB**（企业服务总线）：消息路由、协议转换、服务编排
- 服务契约：WSDL
- 服务注册：UDDI
- 消息协议：SOAP / REST

## SOA vs 微服务

| 维度 | SOA | 微服务 |
|---|---|---|
| 粒度 | 粗 | 细 |
| 通信 | ESB（集中） | 轻量级（去中心化） |
| 数据 | 共享数据库常见 | 每服务独立数据库 |
| 部署 | 重量级应用服务器 | 容器 |

## 案例题常见问法

- ESB 选型与职责
- 服务编排 vs 服务编制（Orchestration vs Choreography）
