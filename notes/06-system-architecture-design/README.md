# 06 · 系统架构设计（教材第 7 章）— 🔥 核心中的核心

## 核心考点

### 1. 架构风格（必考）

- **数据流**：批处理、管道—过滤器
- **调用—返回**：主程序—子程序、面向对象、层次式、客户端—服务器
- **独立构件**：进程通信、事件驱动（发布—订阅）
- **虚拟机**：解释器、规则系统
- **以数据为中心**：数据库仓库、黑板（Blackboard）
- **闭环控制**：过程控制

### 2. 软件架构建模（4+1 视图）

逻辑视图、开发视图、进程视图、物理视图 + 场景（用例）

### 3. 特定领域软件架构 DSSA

三层：领域开发环境 → 领域特定应用开发环境 → 应用执行环境
三角色：领域专家、领域分析师、领域设计师

### 4. 软件产品线

核心资产 + 产品开发 + 管理
