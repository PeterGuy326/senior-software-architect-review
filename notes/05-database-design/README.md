# 05 · 数据库设计（教材第 6 章）

## 核心考点

- 关系代数（选择、投影、连接、除）
- 函数依赖、Armstrong 公理、闭包、候选码求解
- 范式：1NF → 2NF → 3NF → BCNF → 4NF（**高频考点**）
- E-R 图 → 关系模式转换
- 事务 ACID、隔离级别（读未提交/读已提交/可重复读/串行化）
- 并发控制：封锁协议（一级/二级/三级）、两段锁（2PL）
- 数据库恢复：日志、检查点、undo/redo
- 分布式数据库：CAP、BASE、最终一致性
- 数据仓库：OLAP、星型/雪花模型、ETL
- NoSQL：KV、文档、列族、图

## 高频考法

- 案例题：给出 E-R 图要求**转关系模式 + 写出范式判断 + 分解到 3NF**
