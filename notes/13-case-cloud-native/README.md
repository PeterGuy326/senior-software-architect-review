# 13 · 案例：云原生架构（教材第 14 章）— 🔥 近年高频

## 核心概念

- **四大要素**：容器化、微服务、DevOps、持续交付
- **十二要素应用**（12-Factor）
- **微服务拆分**：DDD 限界上下文、业务能力、团队自治（康威定律）

## 常见组件

- 服务注册发现：Eureka / Consul / Nacos
- API 网关：Spring Cloud Gateway / Kong
- 配置中心：Apollo / Nacos
- 熔断限流：Hystrix / Sentinel / Resilience4j
- 链路追踪：SkyWalking / Jaeger / Zipkin
- 容器编排：Kubernetes
- Service Mesh：Istio + Envoy

## 案例题常见问法

- 单体 → 微服务拆分策略
- 服务治理（熔断、降级、限流）
- 分布式事务（TCC / Saga / 本地消息表）
- 多活 / 异地容灾
