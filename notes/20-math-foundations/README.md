# 20 · 数学与运筹学基础（📚 综合 2–3 题）

## 核心考点

### 一、图论

- 最短路径：**Dijkstra**（非负权）、**Floyd**（多源）、**Bellman-Ford**（带负权）
- 最小生成树：**Prim**、**Kruskal**
- 拓扑排序（有向无环图 DAG）
- 关键路径（PERT/CPM，结合项目管理）

### 二、线性规划

- 目标函数 + 约束 + 非负
- 单纯形法求解思路
- 对偶问题

### 三、动态规划

- 最优子结构 + 重叠子问题
- 典型问题：背包、LCS、编辑距离

### 四、排队论

- M/M/1 模型：
  - 平均队长 L = ρ / (1-ρ)
  - 平均等待时间 W = ρ / (μ-λ)
- ρ = λ / μ（利用率）

### 五、概率与统计

- 条件概率、贝叶斯
- 期望、方差
- 正态分布

### 六、决策论

- 期望值决策
- 决策树

## 考频

综合知识每年约 2–3 题，**不是重点，有空再啃**
