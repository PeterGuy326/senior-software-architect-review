# 15 · 案例：嵌入式系统架构（教材第 16 章）

## 核心概念

- 嵌入式系统特性：实时性、资源受限、专用性、可靠性
- 实时性：硬实时 vs 软实时
- RTOS：VxWorks / FreeRTOS / RT-Linux
- 嵌入式架构模式：循环轮询、前后台、事件触发、时间触发
- 调度：RMS（Rate Monotonic Scheduling）、EDF（Earliest Deadline First）

## 典型场景

- 车载系统、工业控制、消费电子、物联网设备

## 案例题常见问法

- 实时性保证
- 功耗优化
- 软硬件协同设计
