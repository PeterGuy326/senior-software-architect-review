# 04 · 软件工程（教材第 5 章）

## 核心考点

- 软件开发模型：瀑布 / 原型 / 螺旋 / V / 增量 / 迭代 / RUP / 敏捷（Scrum、XP、Kanban）
- 需求工程：需求获取、分析（DFD、UML 用例）、规格说明（SRS）、验证
- 软件设计：结构化（模块独立性：耦合/内聚）、面向对象（SOLID）
- 软件测试：单元/集成/系统/验收；黑盒/白盒；静态/动态；McCabe 圈复杂度
- 软件度量：功能点（FP）、代码行（LOC）
- 配置管理、能力成熟度（CMMI 五级）
- 项目管理：PERT、关键路径（CPM）、挣值管理（PV/EV/AC/SV/CV）

## 高频计算

- **关键路径**：从开始到结束耗时最长的路径
- **挣值**：SPI = EV/PV、CPI = EV/AC
