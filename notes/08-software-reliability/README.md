# 08 · 软件可靠性（教材第 9 章）

## 核心考点

- 可靠性指标：MTTF、MTTR、MTBF、可用性 A = MTBF / (MTBF + MTTR)
- 可靠性模型：种子法、失效率曲线（浴盆曲线）
- 容错技术：
  - N 版本程序设计
  - 恢复块（Recovery Block）
  - 防卫式程序设计
- 冗余：
  - 结构冗余（静态/动态/混合）
  - 信息冗余（校验码）
  - 时间冗余（重试）
  - 冗余附加（附加硬件/软件）
